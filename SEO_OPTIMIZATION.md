# SEO Оптимизация - Фиоритура

## Уже реализовано:

### 1. ✅ Meta теги и SEO базовое
- Meta description для правильной выдачи в Google и Яндекс
- Meta keywords для релевантного поиска
- Open Graph теги для красивого отображения в соцсетях (ВКонтакте, Telegram, Facebook)
- Twitter Card метатеги
- Canonical тег (https://fioratura.ru/) чтобы избежать дублей URL

### 2. ✅ Структурированные данные (JSON-LD)
- Разметка локального бизнеса (LocalBusiness) с полной информацией
- Координаты Google Maps (54.671513, 20.474195)
- Разметка услуг (ProfessionalService)
- Информация о педагоге и компании

### 3. ✅ Файлы для поисковиков
- **robots.txt** — на сайте управляет индексацией
- **sitemap.xml** — помогает поисковикам найти все страницы

### 4. ✅ Оптимизация загрузки
- Минифицированный CSS (улучшает скорость и ранжирование)
- Критический CSS (улучшает First Contentful Paint)
- Lazy loading изображений
- Service Worker для оффлайн функциональности
- Gzip сжатие на сервере
- Кэширование браузером

### 5. ✅ Аналитика
- Яндекс.Метрика для отслеживания посещений
- Готово к интеграции Google Analytics

---

## Что нужно сделать вручную:

### 1. Google Search Console
1. Перейти на https://search.google.com/search-console
2. Добавить сайт https://fioratura.ru
3. Выбрать верификацию через meta тег
4. Скопировать код и вставить в index.html вместо `ВСТАВЬТЕ_КОД_GOOGLE_СЮДА`
5. Отправить sitemap.xml в консоль
6. Отправить robots.txt в консоль

### 2. Яндекс Вебмастер
1. Перейти на https://webmaster.yandex.ru
2. Добавить сайт https://fioratura.ru
3. Выбрать верификацию через meta тег
4. Скопировать код и вставить в index.html вместо `ВСТАВЬТЕ_КОД_ЯНДЕКСА_СЮДА`
5. Загрузить sitemap.xml в панель Яндекса
6. Загрузить robots.txt

### 3. Рекомендации по контенту
- Добавьте более описательные заголовки h1 для каждого раздела
- Используйте более длинные, описательные заголовки (H1, H2)
- Добавьте важные ключевые слова в текст (вокальная студия, уроки пения, педагог, Калининград)
- Обновляйте контент регулярно (добавляйте отзывы, новые фото)

### 4. Ссылки и социальные сети
- Получите ссылки с авторитетных сайтов Калининграда
- Добавьте социальные кнопки для ВКонтакте, Instagram, Facebook
- Обновляйте соцсети с новостями и фото

## Файлы в проекте:

